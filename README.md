## 如何运行

### 懒人运行
先确保你电脑上装了git和java, 然后随便找个合适的文件下运行下面的代码:
```
git clone https://github.com/relish-wang/Tetris.git && cd Tetris && java -jar Tetris.jar
```

### macOS/Linux
下面方法选其一即可
 - 1 在项目根目录下打开终端执行
```shell
sh start.sh
```
 - 2  在项目根目录下打开终端执行
```shell
./start.sh # 若遇到": permission denied: " 就先执行`chmod +x ./start.sh`
```
### Windows
下面方法选其一即可
 - 1 双击跟目录下`start.bat`文件
 - 2 在项目根目录下打开终端执行
 ```shell
start start.bat
 ```
